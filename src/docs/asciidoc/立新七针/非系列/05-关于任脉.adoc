== 任脉的别络

任脉的别出络脉，名叫尾翳(本义为羽毛做的华盖)，鸠尾穴的别称：由此别出下行，散于腹
部。如果本络脉发生病变，属实的是腹皮痛，属虚的是谷道搔痒。治疗这些病证，可取本经
别出络穴尾翳。
